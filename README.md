# Deep Learning Engineering

| #    | Наименование проекта                | Описание                                                     | 
| ---- | ------------------------------------------------------------ | ------------------------------------------------------------ | 
| 1.   | [Нейросеть для автодополнения текстов](https://github.com/vvyavorskaya/deep-learning/tree/master/text-autocomplete) | Создание рекуррентной нейросети, которая на основе начала фразы предсказывает её продолжение.  
| 2.   | [Нейросеть для предсказания калорийности блюда](https://github.com/vvyavorskaya/deep-learning/tree/master/calories_counter) | Создание и обучение нейросети, которая будет предсказывать калорийность блюд (точность оценки MAE < 50).
| 3.   | [Pretrain & SFT](https://github.com/vvyavorskaya/deep-learning/tree/master/pretrain-and-sft) | Реализация pretrain и posttrain этапов обучения LLM на базе текстов произведений русской литературы. 
| 4.   | [Retrieval-система по статьям с arXiv](https://github.com/vvyavorskaya/deep-learning/tree/master/arxiv_rag) | Построение retrieval-системы и оценка её качества на базе датасета с аннотациями статей с портала arXiv.org и тестового набора запросов и правильных ответов к ним. Целевая метрика — качество поисковой выдачи: MRR@5 > 0.91.
| 5.   | [Система поиска подходящих товаров на основе дообученной CLIP модели](https://github.com/vvyavorskaya/deep-learning/tree/master/clip) | Разработка системы интеллектуального поиска одежды для крупного международного онлайн-магазина. Целевой показатель: средний CLIP-score на валидации должен быть выше 30.
